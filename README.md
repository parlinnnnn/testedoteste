# WhatsApp Activity Checker Landing Page

This simple static site simulates a tool for viewing public WhatsApp activity.
It contains three pages:

- `index.html` – input form
- `loading.html` – fake progress screen
- `results.html` – mock results (no real data)

All files are self-contained and use vanilla HTML, CSS and JavaScript. To host on
**GitHub Pages**, enable Pages in your repository settings and choose the main
branch as the source. The site will then be available at
`https://<username>.github.io/<repo>/`.
